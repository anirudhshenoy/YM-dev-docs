encodingAndDecodingPayload

Encoding a payload sending it to a bot:#

First, the payload has to be encoded in a base64 format:

let payload = {
"hello" : "world"
};
//the payload can be anything; we've used object as an example here
let encodedPayload = Buffer.from(JSON.stringify(payload)).toString("base64");

Then, append 'ym.payload' param to the botlink and send the encodedPayload, for example:

`https://app.yellowmessenger.com/pwa/mobile/x16**********9?ym.payload=${encodedPayload}`

Decoding the payload sent to a bot:#

The payload will be stored in app.profile.payload. Convert the incoming payload to a string:

let decodedPayload = Buffer.from(app.profile.payload, 'base64').toString();

If your payload is an object, you'll have to parse it:

let parsedPayload = JSON.parse(decodedPayload);
app.log(decodedPayload.hello) //output: world
